# Peace Net 与成熟论坛功能差距分析

## 当前已有功能

- 用户注册/登录
- 新闻发布（管理员）
- 故事发布（用户）
- 点赞、收藏、评论
- 用户个人主页
- 管理后台

## 缺少的功能

### 内容层面

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 分类/板块 | 内容按主题分类，如"个人经历"、"求助信息"、"资源分享" | 高 |
| 标签系统 | 用户可添加标签，便于筛选 | 中 |
| Markdown 编辑器 | 支持富文本格式、图片上传 | 高 |
| @提及 | 在评论中 @其他用户 | 低 |
| 引用回复 | 回复时引用原评论内容 | 低 |
| 全文搜索 | 搜索新闻、故事、评论 | 高 |

### 用户层面

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 关注用户 | 关注其他用户，查看动态 | 中 |
| 私信系统 | 用户间私信交流 | 低 |
| 通知系统 | 点赞、评论、关注等通知 | 高 |
| 用户等级 | 根据活跃度显示等级 | 低 |

### 管理层面

| 功能 | 说明 | 优先级 |
|------|------|--------|
| 版主角色 | 分配版主管理特定板块 | 低 |
| 举报功能 | 用户举报违规内容 | 中 |
| 审核队列 | 待审核内容集中管理 | 中 |
| 敏感词过滤 | 自动过滤敏感词汇 | 中 |
| 数据统计 | 用户活跃度、内容统计 | 低 |

## 建议实施顺序

1. **第一阶段**：Markdown 编辑器 + 图片上传
2. **第二阶段**：分类/板块系统
3. **第三阶段**：通知系统
4. **第四阶段**：全文搜索
5. **后续迭代**：根据用户反馈添加其他功能
